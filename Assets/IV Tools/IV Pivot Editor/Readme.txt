Easily set your pivot points inside Unity3D.
Unlike the other pivot editor solutions on the asset store you can set the pivot point using scene handles, snap the pivot to mesh vertices.
IV Pivot Editor v2

Features:
       • Manipulate pivot using scene handles 
       • Snap pivot to vertex.
       • Set pivot point according to the mesh bounding box.
       • Center pivot (per axis).
       • Freeze mesh rotation (rotate pivot) 
       • Freeze mesh scale 
       • Works with grouped objects.
	   • Set Pivot to custom transform
	   • Set pivot to custom position or rotation
	   • Sets the pivot on an empty object
	   • Helper messags and easy tooltips
       • Extremely fast.
